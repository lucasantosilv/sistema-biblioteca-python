# sistema-biblioteca-python
# 📚 Sistema de Biblioteca em Python

## 📖 Sobre o projeto
O projeto começou como uma forma simples de armazenar informações sobre livros. Durante o desenvolvimento, evoluiu para um sistema com automações utilizando Python e Programação Orientada a Objetos.

Ainda há pontos a serem aprimorados, mas já cumpre seu objetivo inicial e serve como base para futuras melhorias.

---

## 🚀 Funcionalidades

- 📚 Adicionar livros à biblioteca
- 📋 Listar livros disponíveis
- 🔍 Buscar livros por título
- 💻 Interação com o usuário via terminal

---

## 🛠️ Tecnologias utilizadas

- Python
- Programação Orientada a Objetos (POO)

---

## 💡 Conceitos aplicados

- Classes e objetos  
- Encapsulamento  
- Estruturas de dados (listas)
